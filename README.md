 🛠️ NearbyFix – Local Service Provider Platform
 📌 Overview NearbyFix is a full-stack web application that connects users with trusted local service providers based on their PIN code. Users can search for services, view providers, and book appointments through a simple and secure interface. 
 🚀 Features - User Registration and Login - JWT Authentication - PIN Code-Based Service Discovery - Search Local Service Providers - Service Booking System - Secure REST APIs - MongoDB Database Integration - Responsive User Interface 
 🛠️ Tech Stack 
 Frontend - HTML - CSS - JavaScript ### Backend - Node.js - Express.js - REST APIs 
 Database - MongoDB 
 Authentication - JWT (JSON Web Token) 
 Tools - VS Code - Git - GitHub - Postman
